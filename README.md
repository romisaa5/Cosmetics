## 📱 About The App

**Cosmatics** is a modern, full-featured mobile e-commerce application built with Flutter, designed specifically for browsing and purchasing cosmetics and beauty products. It offers a seamless shopping experience with a clean, elegant UI from the very first screen.

---

## ✨ Features

### 🌟 Onboarding & Authentication
| Screen | Description |
|--------|-------------|
| **Splash Screen** | Branded launch screen displayed on app startup |
| **Onboarding** | Multi-step introduction to the app's features |
| **Login** | Secure user sign-in with email & password |
| **Register** | New account creation with user details |
| **Forgot Password** | Request a password reset via email |
| **Reset Password** | Set a new password using a reset token |

### 🛍️ Shopping Experience
| Screen | Description |
|--------|-------------|
| **Home** | Featured products, banners, and recommendations |
| **Categories** | Browse products by beauty category |
| **Cart** | View, update, and manage selected items |
| **Payment** | Secure checkout and payment process |

### 👤 User Account
| Screen | Description |
|--------|-------------|
| **Profile** | View and edit personal account information |

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (>=3.0.0)
- [Dart SDK](https://dart.dev/get-dart)
- Android Studio / VS Code
- An Android or iOS emulator / physical device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/romisaa5/cosmetics.git
   cd cosmetics
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
